# AI Budget Travel Planner – Mobile App

React Native mobile application for the AI-powered travel planning system.

Built using Expo and TypeScript.

---

Overview

The mobile app allows users to:

- Register and authenticate
- Enter trip details and total budget
- Receive AI-generated itineraries
- View budget allocations
- Store and display Trip Pass QR code
- Access itinerary offline (read-only)

---

Tech Stack

- React Native (Expo)
- TypeScript
- React Navigation
- Zustand (state management)
- Axios (API calls)
- Secure storage for JWT tokens
